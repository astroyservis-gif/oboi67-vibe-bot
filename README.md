# 🤖 Автоматизация бизнеса: Бот-помощник для мастера по поклейке обоев

Этот проект — часть моей практики по **вайб кодингу** и автоматизации малого бизнеса. Я создал систему для частного мастера из Смоленска, которая превращает рутинный процесс оценки работ в прозрачный и удобный сервис.

## 🎯 Какую проблему мы решили?
Раньше мастер тратил время на бесконечные переписки и уточнения цен. Теперь бот берет это на себя:
* **Умный калькулятор:** Рассчитывает стоимость исходя из типа обоев, площади и доп. работ.
* **Прогрев клиента:** Демонстрирует экспертность через портфолио и ответы на частые вопросы.
* **Захват лида:** Передает горячую заявку с контактами прямо в Telegram мастеру.
* **Сервис 24/7:** Отвечает клиентам, даже когда мастер на объекте.

## 🛠 Технологический стек
* **Язык:** Python 3.10+
* **Библиотека:** aiogram 3.x (современный асинхронный фреймворк).
* **Архитектура:** FSM (Конечные автоматы) для логики калькулятора.
* **Интеграция:** Планируется связка с n8n для CRM.
